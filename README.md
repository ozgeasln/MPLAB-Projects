# 🚀 MPLAB & Proteus Microcontroller Projects

This repository contains microcontroller-based assignments, laboratory applications, and simulation studies developed during embedded systems and microcontroller programming courses.

The projects focus on low-level programming, hardware interaction, circuit simulation, and embedded system development using MPLAB X IDE and Proteus Design Suite.

---

# 🛠 Technologies & Tools

| Technology | Description |
|---|---|
| MPLAB X IDE | Microcontroller development environment |
| Proteus Design Suite | Circuit simulation and hardware testing |
| C Programming Language | Embedded systems programming |
| Assembly Language | Low-level microcontroller programming |
| Git & GitHub | Version control and repository management |
